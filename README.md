# 📐 AuraNook
**Diseño de planos con alma y precisión.**

AuraNook es una aplicación web intuitiva diseñada para crear espacios profesionales de forma sencilla.

### Características principales:
* 🖋️ **Dibujo fluido:** Creación de muros y estructuras de forma intuitiva.
* 🛋️ **Librería de mobiliario:** Añade elementos para dar escala a tus proyectos.
* 🏢 **Multiplanta:** Gestión de diferentes niveles en un mismo proyecto.
* 📏 **Medición inteligente:** Cálculo automático de áreas y superficies.
* 💾 **Exportación:** Guarda tus progresos y descarga planos listos para compartir.
